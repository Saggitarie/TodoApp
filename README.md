# TodoApp

This is a Basic Todo App created by Kohki Shiga

## Instruction

> git clone

> Install packages using yarn or npm

> Run yarn dev on commandline

## API Endpoints

### User Table
> GET /api/users/ 

> POST /api/users/ 

> GET /api/users/:username 

### Events

> GET /api/event/ 

> POST /api/event/

> DELETE /api/event/:id

> PATCH /api/event/:id
